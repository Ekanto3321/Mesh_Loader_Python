## Instructions

To load your own mesh files:
 - import it into blender and then export it as an .obj file. 

 - use convert.py as 
    - ./convert.py mesh.obj object.py
  
 - finally, just run main.py as 
    - ./main.py

Controls: 
  - Use the arrow keys to rotate manually  
  - Space to Pause/Play rotation
  - Q/E to shift along Y-axis
  - W/S to shift along Z-axis
  - A/D to shift along X-axis 
   
Axis Reference: 

           Y
           ^  / Z
           | /      
           |/
    <-------------- > X 
          /| 
         / |
        /  v
